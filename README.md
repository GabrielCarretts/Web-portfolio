# Personal Web Portfolio and Resume

## Project Overview
A responsive, web-based professional portfolio developed to showcase foundational web development skills. This project serves as a digital resume, translating a traditional CV into an interactive and accessible web format.

## Technical Architecture
This project was built with a strict focus on clean logic, semantic document structure, and responsive design principles. 

### Core Technologies
* **HTML5:** Utilized semantic tagging to ensure accessibility, proper indexing, and clear chronological document flow.
* **CSS3:** * Implemented **Flexbox** for efficient, one-dimensional component alignment.
    * Utilized **CSS Grid** for structural, two-dimensional page layouts.
    * Applied **Media Queries** to construct a fully responsive user experience across desktop, tablet, and mobile environments.

### Features & Integrations
* **Version Control Architecture:** Maintained via Git, utilizing professional branching, tagging (e.g., version archiving), and merging workflows.
* **External API Routing:** Configured a direct communication pipeline utilizing the WhatsApp `wa.me` API for single-click user engagement.

## Development Methodology
The primary objective of this repository is to demonstrate a practical understanding of fundamental Information Systems concepts, specifically the separation of concerns (isolating HTML structure from CSS presentation) and the implementation of logical, scalable styling rules.

## Future Roadmap
To continuously improve the user experience and demonstrate advanced frontend capabilities, the following technical integrations are planned for future deployment:

* **Theme Toggling (Dark Mode):** Implementation of a JavaScript-driven toggle button. This will utilize DOM manipulation and event listeners to dynamically swap CSS variables (custom properties), allowing users to control their preferred visual theme.
* **Interactive Animations:** Integration of subtle CSS transitions and keyframe animations to enhance UI responsiveness, element states, and overall visual hierarchy without compromising load times.

## Usage
To view this project locally:
1. Clone the repository to your local environment.
2. Navigate to the root directory.
3. Open `index.html` in any standard web browser.
